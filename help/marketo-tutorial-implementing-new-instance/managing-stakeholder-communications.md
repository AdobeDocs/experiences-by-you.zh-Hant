# 管理利害關係人以實施Marketo Engage

實作Marketo Engage是擴充您的MarTech棧疊的關鍵時刻。 從行銷到銷售到IT，您可能要將不同的利害關係人納入其中。 瞭解如何透過提出正確的問題並定期溝通所需的更新和支援，從您的組織獲得新Marketo Engage執行個體的支援。 使用教學課程和範本（具有可下載的版本）在整個實施和使用者上線過程中引導您的內部通訊。

## 如何與利害關係人互動以及您的Marketo Engage實作領導力

### 動作1 — 配合領導層和利害關係人，將Marketo Engage功能與其目標聯絡起來
